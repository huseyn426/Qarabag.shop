# 🏟️ Qarabag Shop — Premium Fanshop

A modern, high-performance web store for fans of **Qarabag FK**, built with pure JavaScript for maximum speed and efficiency.

## 🚀 Key Features

* **Dynamic Product Grid:** Products are rendered on-the-fly from local storage.
* **Smart Filtering:** Instant category switching (Hoodies, T-shirts, Scarfs, etc.) with active state highlighting.
* **Multilingual Support:** Fully functional translation system for **Azerbaijani (AZ)**, **Russian (RU)**, and **English (EN)**.
* **Advanced Admin Panel:** A secure interface to manage the inventory without a backend.
* **Image Optimization:** Built-in **Canvas API** compression that automatically resizes uploaded photos to prevent `LocalStorage` quota errors.
* **Fully Responsive:** Optimized for both desktop and mobile devices.

## 🛠️ Tech Stack

* **HTML5 / CSS3** (Flexbox, Grid, AOS Animations)
* **JavaScript (ES6+)**
* **LocalStorage** (Client-side database)
* **Canvas API** (Image processing & compression)
